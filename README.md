Calorias de mi Comida
Es un proyecto con fin educativo, informativo y de entretenimiento.
Su funcion principal es calcular las calorias de cada comida en especifico.

Funciona mediante un Backend el cual integra la API de Gemini IA, la cual se encarga de realizar la busqueda inteligente en bases de datos nutricionistas certificadas.
Se agrega una base de datos para guardar resultados de busqueda, de esta manera se pueden ahorrar tokens cuando el proyecto cresca y las personas
empiezen a realizar consultas repetitivas.

La magia del proyecto ocurre al recibir los datos limpios en el frontend, despues de escribir el texto en lenguage natural, es decir; si un usuario ingresa el texto: me comí 150 gramos de pescado, y 50 de arroz,
la respuesta que recibira sera el total de calorias que consumio por cada producto, y un parrafo educativo sobre esa comida en especifico.

Se empleo una arquitectura módular, con separacion de responsabilidades bucando siempre seguir los mejores estandares de código limpio
